# API for Sales

An e-commerce sales API. This API has features such as login, session creation, product registration, user registration, listing, authentication middleware and caching service.

## Technologies
- ✔️ Express
- ✔️ TypeORM
- ✔️ JsonWebToken
- ✔️ BcryptJS
- ✔️ Multer
- ✔️ Redis
- ✔️ PostgresSQL
